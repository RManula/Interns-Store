# Data Engineering Internship - NAB

## Role Description
NAB's Data Engineering team manages the enterprise data platform that powers NAB's analytics, AI, regulatory reporting, and customer personalisation capabilities across Australia and internationally. As a Data Engineering intern, you'll contribute to the pipelines and data infrastructure that process billions of banking events daily, working with modern cloud-native data tooling on AWS.

## Key Responsibilities
* Build and optimise ETL/ELT data pipelines using Apache Spark, dbt, and Airflow on NAB's AWS data platform
* Implement data quality monitoring, schema validation, and lineage tracking for critical data domains (customer, transaction, risk)
* Collaborate with data scientists, risk modellers, and analytics teams to understand data requirements and engineer appropriate datasets
* Contribute to NAB's data governance framework including metadata cataloguing and data domain documentation

## Technical Skills & Requirements
* Strong SQL skills with experience working on large-scale transactional or event-driven datasets
* Python proficiency for pipeline development and data transformation scripting
* Familiarity with cloud data platforms (AWS preferred: Redshift, Glue, S3, EMR)
* Enrolled in Computer Science, Data Engineering, Information Systems, or Software Engineering (penultimate or final year)

## Perks & Benefits
* Hands-on experience with enterprise-grade data engineering at one of Australia's largest data estates
* AWS Data Engineer or Databricks Associate certification voucher supported by NAB
* Access to NAB's data platform architecture documentation and internal data engineering best practice guides
* Priority pathway to NAB's Data Engineering and Analytics Graduate Program
