# Software Engineering Internship - NAB

## Role Description
NAB's Technology & Engineering division employs thousands of technologists building and maintaining the digital banking platforms used by over 9 million Australians. As a Software Engineering intern, you'll join a product engineering squad delivering microservices, APIs, and digital features for NAB's mobile app, NAB Internet Banking, or NAB Connect business banking platform.

## Key Responsibilities
* Design and implement backend microservices or frontend features within NAB's cloud-native banking platform (AWS)
* Write unit, integration, and end-to-end automated tests as part of NAB's test-driven development culture
* Participate in agile ceremonies and contribute to NAB's engineering community of practice
* Develop API integrations using OpenAPI specifications and contribute to NAB's internal API marketplace

## Technical Skills & Requirements
* Proficiency in Java (Spring Boot), Python, or TypeScript for backend or full-stack development
* Understanding of RESTful API design, microservices patterns, and AWS cloud services
* Experience with Git workflows, CI/CD pipelines, and containerisation (Docker, Kubernetes)
* Enrolled in Computer Science, Software Engineering, or Information Technology (penultimate or final year)

## Perks & Benefits
* Contribute code to banking platforms used by millions of Australians daily
* NAB Tech Academy access covering AWS, DevOps, and software engineering learning paths
* Hybrid working with 3 days at NAB's tech-forward Docklands headquarters in Melbourne
* Strong graduate conversion pipeline into NAB's Software Engineering and Cloud Engineering Graduate streams
