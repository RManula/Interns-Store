# Software Engineering Internship - CommBank

## Role Description
CommBank's Engineering & Technology division employs over 8,000 technologists making it one of Australia's largest technology employers. As a Software Engineering intern, you'll join a product engineering squad building microservices, APIs, or customer-facing features within CBA's NetBank or CommBiz platforms, contributing meaningful code to systems used by millions of Australians daily.

## Key Responsibilities
* Develop and test new features across backend microservices (Java/Spring Boot) or frontend web components (React/TypeScript)
* Write unit, integration, and regression tests as part of a test-driven development (TDD) culture
* Participate in agile ceremonies including daily stand-ups, sprint planning, retrospectives, and code reviews
* Contribute to internal documentation, technical design records, and API specification updates

## Technical Skills & Requirements
* Proficiency in at least one of: Java, Python, TypeScript, or Kotlin
* Understanding of RESTful API design, microservices architecture, and containerisation (Docker)
* Experience with Git-based workflows and familiarity with CI/CD pipelines (Jenkins, GitHub Actions)
* Enrolled in Computer Science, Software Engineering, IT, or related technical degree (penultimate or final year)

## Perks & Benefits
* Hybrid work model — 3 days per week in CommBank's modern Sydney CBD tech hub
* Access to Pluralsight, O'Reilly Learning, and internal CBA tech training library
* Subsidised canteen, end-of-trip cycling facilities, and employee health & wellness programs
* Strong graduate conversion pathway into CBA's Software Engineering Graduate stream
