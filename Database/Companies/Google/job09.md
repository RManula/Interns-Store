# Cybersecurity Internship - Google

## Role Description
Google's Security Engineering team in Sydney sits within the broader Google Trust & Safety and GCAT (Google Cloud Advisory Team) organisations. As a Cybersecurity intern, you'll contribute to threat detection, vulnerability research, and security automation initiatives that protect Google's infrastructure and its enterprise customers across APAC.

## Key Responsibilities
* Conduct vulnerability assessments and security code reviews across internal tools and third-party integrations
* Develop security automation scripts and SIEM detection rules to identify anomalous activity in large-scale log environments
* Participate in red/blue team exercises, tabletop simulations, and security engineering reviews
* Research emerging threats relevant to cloud environments and document findings as internal threat intelligence reports

## Technical Skills & Requirements
* Foundational knowledge of cybersecurity concepts: threat modelling, OWASP Top 10, network security, and cryptography
* Programming skills in Python for scripting security automation and analysis tools
* Familiarity with security tools (Burp Suite, Wireshark, SIEM platforms) is highly regarded
* Pursuing a degree in Cybersecurity, Computer Science, Information Systems Security, or equivalent (penultimate or final year preferred)

## Perks & Benefits
* Sponsored GIAC or CompTIA Security+ exam voucher upon successful completion of internship
* Weekly security-focused knowledge-sharing sessions with Google's global security teams
* Access to Google's internal threat intelligence platform and CTF competition infrastructure
* Security clearance pathway support for candidates interested in government-adjacent security roles post-graduation
