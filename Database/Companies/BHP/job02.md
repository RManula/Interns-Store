# Data Engineering Internship - BHP

## Role Description
BHP's Global Data & Analytics Centre in Brisbane powers the data infrastructure supporting operations across BHP's copper, iron ore, nickel, and potash assets globally. As a Data Engineering intern, you'll join a team building and maintaining the pipelines, data models, and platforms that turn raw operational sensor data into trusted, analytics-ready datasets for BHP's decision-makers and AI models worldwide.

## Key Responsibilities
* Build and maintain industrial IoT data ingestion pipelines processing real-time sensor data from mine sites, processing plants, and port operations
* Transform raw operational datasets into clean, trusted tables in BHP's cloud data platform (Azure Data Lake, Databricks) using dbt and Spark
* Implement data quality monitoring, schema validation, and pipeline observability solutions
* Collaborate with data scientists and operational technology teams to understand data requirements and deliver fit-for-purpose data products

## Technical Skills & Requirements
* Strong Python and SQL skills for data pipeline development and large-scale data transformation
* Familiarity with cloud data platforms (Azure preferred: ADLS Gen2, Azure Data Factory, Databricks, Synapse Analytics)
* Understanding of data pipeline orchestration tools (Airflow, Azure Data Factory) and streaming concepts (Kafka, Event Hubs)
* Enrolled in Computer Science, Data Engineering, Software Engineering, or Information Systems (penultimate or final year)

## Perks & Benefits
* Work with globally unique mining operational datasets from some of the world's largest mine sites
* Azure Data Engineer Associate certification voucher provided and supported
* BHP Brisbane office in the CBD with flexible hybrid working arrangements and modern collaborative facilities
* Pathway to BHP's Data Engineering Graduate track within the Global Data & Analytics Centre
