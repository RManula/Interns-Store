# 💡 Team Ideas Board

This is the space for **everyone** to add ideas, features, and suggestions for their section.
Don't hold back — half-formed ideas are welcome, we refine them together.

## Where to add your idea

Open the file for your section and add your idea at the bottom:

| File | Section | Owner |

| [`02-internships-and-search.md`](02-internships-and-search.md) | Internships & Search | Member 2 |

Got an idea that doesn't fit your section? Add it anyway in the closest file and tag who it's for.

## How to add an idea

Copy this block into your section file and fill it in:

```markdown
### <short title for the idea>
- **Feature:** what it is, in one or two lines
- **Why it helps:** the problem it solves, or the marks it earns
- **How it should be built:** the pages/screens, components, or behaviour you imagine
- **Priority:** MUST / REAL / ASPIR
- **Added by:** <your name>
```

## Priority tags

- **MUST** — required for the assignment/marks. Non-negotiable.
- **REAL** — realistic and buildable for this project now.
- **ASPIR** — aspirational stretch goal; only if time allows.

## Two ways to contribute

1. **Quick edit:** open the file on GitHub → ✏️ → paste your idea → commit.
2. **Pull request:** create a branch, add your ideas, open a PR for the team to review.
